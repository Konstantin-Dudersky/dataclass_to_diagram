# kroki-python
Lib for interaction with https://kroki.io

## Запустить тест:
```sh
poetry run poe docs
```

Или запусить task в vs code - F1 -> Task: Run task -> docs

## Загрузить пакет в pypi

Собрать пакет
```sh
poetry build
```

Опубликовать пакет
```sh
poetry publish
```
